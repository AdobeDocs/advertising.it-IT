# [!UICONTROL Google Performance Max Placement Report]

*Applicabile solo a [!DNL Google Ads] account con campagne con prestazione massima*

[!UICONTROL Google Performance Max Placement Report] include i dati sulle impression per i posizionamenti nelle campagne con prestazione massima di [!DNL Google Ads] all&#39;interno di un account specificato. Il rapporto include le impression per ogni posizionamento incluso per URL di destinazione. Per impostazione predefinita, i dati includono una riga per ogni combinazione di URL di posizionamento e destinazione che ha ricevuto almeno un’impression nell’intervallo di dati specificato, anche se il posizionamento è stato eliminato durante il periodo di reporting. Per impostazione predefinita, le righe sono in ordine crescente per campagna.

Utilizzare questo report per:

* identifica i siti web, le app e i canali [!DNL YouTube] che servono i tuoi annunci.

* identificare ed escludere posizionamenti irrilevanti o con prestazioni insoddisfacenti per migliorare il ROI delle campagne.

* crea solidi elenchi di esclusione per garantire che gli annunci vengano visualizzati solo in ambienti brand-safe.
