# (Nuova interfaccia) Informazioni sulla visualizzazione [!UICONTROL Keywords]

*funzionalità Beta*

La visualizzazione [!UICONTROL Manage] > [!UICONTROL Keywords] include due schede:

* Nella scheda [!UICONTROL Keywords] sono elencate tutte le parole chiave nella visualizzazione filtrata per l&#39;account dell&#39;inserzionista selezionato.

* Nella scheda [!UICONTROL Negatives] sono elencate tutte le parole chiave negative nella visualizzazione filtrata per l&#39;account dell&#39;inserzionista selezionato.

* [Assegna classificazioni etichette](/help/search-social-commerce/new-ui/reports/label-classifications-manage.md) alle parole chiave
