# (Nuova interfaccia) Informazioni sulla visualizzazione [!UICONTROL Placements]

*funzionalità Beta*

La visualizzazione [!UICONTROL Manage] > [!UICONTROL Placements] include due schede:

* Nella scheda [!UICONTROL Placements] sono elencati tutti i posizionamenti nella visualizzazione filtrata per l&#39;account dell&#39;inserzionista selezionato.

* Nella scheda [!UICONTROL Negatives] sono elencati tutti i posizionamenti negativi nella visualizzazione filtrata per l&#39;account dell&#39;inserzionista selezionato.
