# (Nuova interfaccia) Visualizzare i registri della cronologia delle modifiche

Il report [!UICONTROL History Logs] include un registro delle modifiche apportate all&#39;account dell&#39;inserzionista negli ultimi 31 giorni. Il rapporto può includere modifiche ai seguenti tipi di oggetti: utenti (inserzionisti), portfolio, campagne, gruppi di annunci, annunci, parole chiave, posizionamenti e target di prodotto. Puoi ordinare e filtrare i dati in base a qualsiasi colonna.

È possibile scaricare ulteriori informazioni sui registri della cronologia dell&#39;inserzionista in un file in formato cartella di lavoro [!DNL Microsoft Excel] (file XLSX). Il rapporto include i valori vecchi e nuovi e l’ora in cui si è verificata la modifica.

>[!NOTE]
>
>Per le modifiche ai portfolio, vedere anche la cronologia delle modifiche del [portfolio](/help/search-social-commerce/new-ui/manage/portfolios/portfolio-view-change-history.md).

## Visualizza il report [!UICONTROL History Logs] {#history-logs-open}

1. Nel menu principale, fare clic su **[!UICONTROL Reports]** > **[!UICONTROL History Logs]**.

1. (Facoltativo) [Modifica i dati inclusi nella visualizzazione](/help/search-social-commerce/common-tasks/data-views/data-views-about.md).

## Gestisci download report per [!UICONTROL History Logs]

### Generare un rapporto con le righe di dati filtrate

1. [Aprire i registri della cronologia](#history-logs-open).

1. In alto a destra, fai clic su ![Scarica report](/help/search-social-commerce/assets/download.png "Scarica report").

1. Nelle impostazioni di [!UICONTROL Grid Reports], immettere un nome di report univoco, quindi fare clic su **[!UICONTROL Generate]**.

   Per impostazione predefinita, il file è denominato &quot;allChangeHistory_YYYYMMDD_NNNN&quot;, dove &quot;NNNN&quot; è il numero del processo sequenziale (ad esempio &quot;allChangeHistory_20260402_1326).

   Il file viene aggiunto all&#39;elenco [!UICONTROL Recently Generated].

1. (Facoltativo) Per scaricare il file una volta completato, fai clic su ![Scarica](/help/search-social-commerce/assets/download.png "Scarica") accanto al nome del file.

   Il file viene scaricato in base alla normale procedura del browser.

### Scaricare un rapporto completato

1. [Aprire i registri della cronologia](#history-logs-open).

1. In alto a destra, fai clic su ![Scarica report](/help/search-social-commerce/assets/download.png "Scarica report").

1. Nell&#39;elenco [!UICONTROL Recently Generated] della finestra di dialogo [!UICONTROL Grid Reports], fare clic su ![Scarica](/help/search-social-commerce/assets/download.png "Scarica") accanto al nome del file.

   Il file viene scaricato in base alla normale procedura del browser.

### Eliminare un rapporto completato

1. [Aprire i registri della cronologia](#history-logs-open).

1. In alto a destra, fai clic su ![Scarica report](/help/search-social-commerce/assets/download.png "Scarica report").

1. Nell&#39;elenco [!UICONTROL Recently Generated] della finestra di dialogo [!UICONTROL Grid Reports], fare clic su ![Elimina](/help/search-social-commerce/assets/delete-new.png "Elimina") accanto al nome del file.
