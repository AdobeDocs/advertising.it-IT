# (Nuova interfaccia) Visualizzare la cronologia delle modifiche di un portfolio

*funzionalità Beta*

Puoi visualizzare un registro delle modifiche apportate al portfolio durante qualsiasi intervallo di date. Le azioni registrate includono la creazione del portfolio, tutte le modifiche alle impostazioni del portfolio, tutte le campagne assegnate o non assegnate al portfolio, tutti i vincoli applicati o rimossi dalle unità di offerta del portfolio, tutti i vincoli sulle unità di offerta del portfolio scadute e altre azioni che influiscono sul portfolio

È possibile filtrare i dati per categoria<!-- Not available as of 5-14: and by the user who performed the action -->. È inoltre possibile ordinare l’elenco in base alla data in ordine crescente o decrescente.

Puoi anche scaricare i dati in un file di foglio di calcolo.

## (Nuova interfaccia) Visualizzare la cronologia delle modifiche di un portfolio {#portfolio-change-history}

Le informazioni per ogni modifica includono la data, il nome utente della persona che ha apportato la modifica, il tipo di modifica e il valore modificato.

1. Nel menu principale, fare clic su **[!UICONTROL Goals]>[!UICONTROL Conversions]**.

1. Posizionare il cursore sulla riga del portfolio e fare clic su **[!UICONTROL ...]>[!UICONTROL Change History]**.

1. (Facoltativo) Modifica le informazioni visualizzate:

   * Fai clic su ![Calendario](/help/search-social-commerce/assets/calendar-new.png "Calendario") per modificare l&#39;intervallo di date per il rapporto.

   * Fai clic su ![Filtro](/help/search-social-commerce/assets/filter-new.png "Filtro") per filtrare i dati in base al tipo di modifica.<!-- Not available as of 5-14: and by the user who performed the action -->.

   * Fai clic su ![Ordina](/help/search-social-commerce/assets/sort.png "Ordina") per ordinare l&#39;elenco in base alla data in ordine crescente o decrescente.


## Gestione dei report sui dati delle prestazioni per le cronologie delle modifiche

È possibile scaricare ulteriori informazioni sulla cronologia delle modifiche per un portfolio in un file in formato cartella di lavoro [!DNL Microsoft Excel] (file XLSX). Il rapporto include l’ID di modifica numerico, i valori vecchio e nuovo, il tipo di modifica (ad esempio &quot;aggiunto&quot; o &quot;modificato&quot;), il tipo di azione (ad esempio &quot;Budget&quot; o &quot;Nome Portfolio&quot;) e il sottotipo, se applicabile, l’ora e il nome utente.

### Generare un rapporto con le righe di dati filtrate

1. [Aprire la cronologia modifiche del portfolio](#portfolio-change-history).

1. In alto a destra, fai clic su ![Scarica report](/help/search-social-commerce/assets/download.png "Scarica report").

1. Nelle impostazioni di [!UICONTROL Grid Reports], immettere un nome di report univoco, quindi fare clic su **[!UICONTROL Generate]**.

   Per impostazione predefinita, il file è denominato &quot;portfolioChangeHistory_YYYYMMDD_NNNN&quot;, dove &quot;NNNN&quot; è il numero del processo sequenziale (ad esempio &quot;portfolioChangeHistory_20260402_1326).

   Il file viene aggiunto all&#39;elenco [!UICONTROL Recently Generated].

1. (Facoltativo) Per scaricare il file una volta completato, fai clic su ![Scarica](/help/search-social-commerce/assets/download.png "Scarica") accanto al nome del file.

   Il file viene scaricato in base alla normale procedura del browser.

### Scaricare un rapporto completato

1. [Aprire la cronologia modifiche del portfolio](#portfolio-change-history).

1. In alto a destra, fai clic su ![Scarica report](/help/search-social-commerce/assets/download.png "Scarica report").

1. Nell&#39;elenco [!UICONTROL Recently Generated] della finestra di dialogo [!UICONTROL Grid Reports], fare clic su ![Scarica](/help/search-social-commerce/assets/download.png "Scarica") accanto al nome del file.

   Il file viene scaricato in base alla normale procedura del browser.

### Eliminare un rapporto completato

1. [Aprire la cronologia modifiche del portfolio](#portfolio-change-history).

1. In alto a destra, fai clic su ![Scarica report](/help/search-social-commerce/assets/download.png "Scarica report").

1. Nell&#39;elenco [!UICONTROL Recently Generated] della finestra di dialogo [!UICONTROL Grid Reports], fare clic su ![Elimina](/help/search-social-commerce/assets/delete-new.png "Elimina") accanto al nome del file.
