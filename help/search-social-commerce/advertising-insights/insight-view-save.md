# Visualizza o salva [!DNL Advertising Insight]

1. Nel menu principale, fare clic su **[!UICONTROL Search, Social, & Commerce]> [!UICONTROL Insights & Reports] >[!UICONTROL Advertising Insights]**.

2. Nel menu a sinistra, fai clic su ![Report](/help/search-social-commerce/assets/insight-reports.png "Report").

3. Nel pannello [!UICONTROL Generated Insights] eseguire una delle operazioni seguenti:

   * Fai clic sull’istanza del rapporto.

   * Posizionare il cursore sull&#39;istanza del report, quindi fare clic su ![Scarica](/help/search-social-commerce/assets/insight-download.png "Scarica").

4. Apri o salva il file in base alla normale procedura del browser.

   Se il file è compresso, estrarre i singoli file in una cartella prima di visualizzarli.
